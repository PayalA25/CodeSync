# CodeSync

**CodeSync** is a real-time collaborative code editor that allows multiple users to join a shared room and write code together with instant synchronization. The application is designed for collaborative development, technical interviews, and live coding sessions.

---

## Features

- Real-time code synchronization across all connected users
- Room-based collaboration using unique room IDs
- Low-latency communication with WebSockets
- Syntax-highlighted code editor
- Live user join and leave updates
- Scalable real-time architecture

---

## How It Works

Users create or join a room using a unique room ID. Once connected, any code change made by a user is instantly synchronized and reflected for all other users in the same room. The system uses event-driven communication to maintain consistency across all clients.
